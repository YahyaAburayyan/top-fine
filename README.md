# TopFine Website 🧻

A modern, responsive web application for TopFine - a Palestinian brand specializing in toilet and fine papers. This project was developed as a client website to establish and document the brand's digital presence.

## 🚀 Live Demo
**Website is live and deployed at:** [www.topfine.shop](https://www.topfine.shop)

## 🛠️ Tech Stack

- **Frontend Framework:** React 19.1.1 with TypeScript
- **Build Tool:** Vite 7.1.2
- **Styling:** Tailwind CSS 3.4.17 with RTL support
- **Internationalization:** i18next with browser language detection
- **Development:** ESLint for code quality
- **Deployment:** GitHub Pages with custom domain

## ✨ Features

- 🌐 **Multi-language Support**: Arabic and English with automatic language detection
- 📱 **Responsive Design**: Optimized for all devices
- ↔️ **RTL Support**: Full right-to-left text support for Arabic
- 🎨 **Modern UI**: Clean, professional design with Tailwind CSS
- ⚡ **Fast Performance**: Built with Vite for optimal loading speeds
- 🔧 **Type Safety**: Full TypeScript implementation
- 🌍 **Custom Domain**: Deployed at www.topfine.shop

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YahyaAburayyan/top-fine.git
cd top-fine
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

## 📝 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
